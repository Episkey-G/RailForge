# RF Status

Use this skill when you need the current workflow state without changing anything.

- Entry point: `railforge status`
- Report the last known phase, blockers, and next action.
- Keep the output operational rather than narrative.
- Use this skill before deciding whether to research, plan, execute, or resume.
